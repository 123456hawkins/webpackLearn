### 总结

#### 1.两种开发模式
- 开发模式:代码能编译自动化允许
- 生产模式:代码编译优化输出

#### 2.webpack基本功能
- 开发模式:可以编译ES Module语法
- 生产模式:可以编译ES Module语法,压缩js代码

#### 3.Webpack配置文件
- 5个核心概念:entry(入口文件),output(输出文件),loader(将资源文件转换为模块),plugin(插件),mode(模式)
- devServer(配置开发模式下的配置,如地址,端口号等)

#### 4.WebPack脚本指令语法
- npx webpack打包
- npx webpack serve启动开发服务器
